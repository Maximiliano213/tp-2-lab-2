Trabajo Práctico N°1 — Introducción a Python con Menús

Asignatura: Laboratorio de Aplicaciones II - 6° G

Archivo principal: tp1.py

Descripción: Programa en Python que implementa un menú principal con submenús (Partes A–E). Cada submenú contiene las opciones solicitadas en el enunciado: ejemplos de manejo de variables, control de flujo, funciones, estructuras de datos y POO.

Contenido del repositorio

tp1.py — Código con el menú principal y todos los ejercicios implementados.

README.md — Este archivo con instrucciones.

Cómo ejecutar

Asegurate de tener Python 3 instalado.

Abrí la terminal en la carpeta donde esté tp1.py.

Ejecutá:

python tp1.py

Luego seguí las indicaciones del menú en pantalla.

Descripción breve de cada ejercicio

Parte A — Hola Mundo y Variables

Muestra "Hola Python" en consola.

Declara variables de distintos tipos con comentarios y las muestra.

Muestra el tipo de cada variable usando type().

Parte B — Control de Flujo 4. Determina si un número es par o impar. 5. Clasifica una edad en Niño (0–11), Adolescente (12–17) o Adulto (18+).

Parte C — Funciones 6. Función que calcula el área de un círculo dado su radio. 7. Función que recibe una lista de números y devuelve la suma.

Parte D — Estructuras de Datos 8. Crea una lista de lenguajes y la recorre con for. 9. Usa un diccionario para representar un usuario (nombre, edad, email).

Parte E — Clases y POO 10. Define la clase Persona con propiedades nombre y edad y método saludar(). 11. Crea una instancia de Persona y ejecuta saludar().
